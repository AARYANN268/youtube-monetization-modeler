# youtube-monetization-modeler
This project predicts YouTube ad revenue for individual videos using a Linear Regression model based on video performance and contextual features. It helps content creators and media companies estimate potential earnings and make better content and business decisions.
A Streamlit web application is built to allow users to input video metrics and instantly estimate potential ad revenue.
